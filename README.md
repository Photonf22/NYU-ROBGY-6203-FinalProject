# NYU-ROBGY-6203-FinalProject
Final project for ROBGY6203

Please add vis_nav_player as a submodule in this repository
run "git submodule add git@github.com:ai4ce/vis_nav_player.git"

Add the files inside of the folder ""src"" called "dp_improved.py" inside of the vis_nav_player folder
Add the unzipped folder data.zip inside of the vis_nav_player folder and unzip it.
Add "startup.json " inside of the unzip data folder. It is needed since this is the newest updated startup.json file given to us

Run the file "dp_improved.py" with the following 
1) "conda activate game"
2) python3.10 dp_improved.py and see the file run
